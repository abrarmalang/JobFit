LLM Configuration

Configure LLM providers for intelligent CV skill extraction

settings

Configuration Methods

check_circle Method 1: Environment Variables (Recommended)

Set API keys as environment variables before starting the server: 

export GROQ_API_KEY="your-groq-api-key-here"
export GEMINI_API_KEY="your-gemini-key-here"
export OPENAI_API_KEY="your-openai-key-here"
export ANTHROPIC_API_KEY="your-anthropic-key-here"

# Then start the server
python3 -m uvicorn webapp.app.main:app --reload

The system will automatically detect and use the configured provider.

description Method 2: Config File

Create config/llm_config.json in your project directory: 

{
  "provider": "groq",
  "api_key": "your-api-key-here",
  "model": "llama-3.1-70b-versatile",
  "temperature": 0.1,
  "max_tokens": 1000,
  "enable_fallback": true
}

The config manager will load this file automatically on startup.

Supported LLM Providers

Groq

RECOMMENDED

Free, ultra-fast inference with Llama models. Generous rate limits.

Environment Variable: GROQ_API_KEY
Config Provider: "groq"
Recommended Model: "llama-3.1-70b-versatile"
Get API Key: console.groq.com

Google Gemini

FREE

Free tier with 60 requests/minute. Good quality for most use cases.

Environment Variable: GEMINI_API_KEY
Config Provider: "gemini"
Recommended Model: "gemini-pro"
Get API Key: Google AI Studio

OpenAI

$5 FREE CREDIT

GPT-3.5-turbo is very affordable (~$0.002 per CV). $5 free credit for new accounts.

Environment Variable: OPENAI_API_KEY
Config Provider: "openai"
Recommended Model: "gpt-3.5-turbo"
Get API Key: platform.openai.com

Anthropic Claude

Claude 3 Haiku is fast and affordable. Limited free credits available.

Environment Variable: ANTHROPIC_API_KEY
Config Provider: "anthropic"
Recommended Model: "claude-3-haiku-20240307"
Get API Key: console.anthropic.com

Hugging Face

FREE

Free tier available but slower and rate-limited. Good for testing.

Environment Variable: HF_API_KEY
Config Provider: "huggingface"
Recommended Model: "mistralai/Mistral-7B-Instruct-v0.2"
Get API Key: huggingface.co/settings/tokens

info

Fallback Mode

If no LLM provider is configured, the system automatically uses rule-based skill extraction:

  • Extracts skills from dedicated skills sections
  • Pattern matching for common skills across all industries
  • Works for tech and non-tech CVs
  • No API costs or rate limits

Note: LLM-based extraction provides better accuracy and works for any profession (healthcare, education, sales, etc.), while fallback mode is faster but less comprehensive.
